# Game of Life

This is an implementation of John Conway's [Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) in Python, using Qt/PySide2.

## Requirements
Python3
pipenv

# Setup & Build
`git clone https://github.com/ajhenri/game-of-life.git`  
`cd game-of-life`  
`pipenv shell`  
`pipenv install`  
`python3 app.py`
